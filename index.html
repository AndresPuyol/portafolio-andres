Agrotech

Proyecto Formativo SENA | Full-Stack

Agricultura Tech

Agrotech es un proyecto que integra tecnología en la agricultura para optimizar el monitoreo de cultivos y la gestión de recursos. Incluye módulos para registro de cultivos, análisis de datos y una plataforma web intuitiva.

Mi Rol: Diseñé la interfaz en Figma, desarrollé el API REST con NestJS para gestionar los productos, construí el frontend con React para consumir los datos y administré la base de datos en pgAdmin.

Backend & DB

NestJS, Node.js, PostgreSQL (pgAdmin).

Frontend & UX

React.js, Figma, HTML5, CSS3.
