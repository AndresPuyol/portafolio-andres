1. Estructura del Proyecto

El proyecto sigue una arquitectura limpia separando responsabilidades: 

/
├── index.html            // Estructura DOM principal
├── css/
│   └── styles.css        // Estilos custom (glassmorphism, etc)
├── js/
│   ├── script.js         // Lógica, gráficos y modales
│   └── tailwind.config.js // Configuración del tema
└── assets/               // Imágenes y recursos

2. Sistema de Animaciones

Desglose de las interacciones implementadas para mejorar la experiencia de usuario (UX).

Background Blobs

Esferas de color difuminadas en el fondo que se mueven suavemente. Implementadas con keyframes CSS personalizados en Tailwind config.

Infinite Carousel

Cinta de tecnologías que se desplaza infinitamente. Usa duplicación de contenido y traslación CSS lineal.

Liquid Hover

Efecto de llenado en las tarjetas de habilidades. Un pseudo-elemento crece de 0% a 100% de altura al pasar el mouse.

Breathing Chart

El gráfico de radar "respira" mediante un script JS que oscila la tensión de las líneas cada 50ms.

3. Stack Tecnológico

  • HTML5: Semántica y estructura accesibles.
  • Tailwind CSS: Estilado utilitario y sistema de diseño.
  • JavaScript: Lógica de interacción y manipulación del DOM.
  • Chart.js: Visualización de datos (Radar).
